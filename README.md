# Linux Admin Basics

This repository contains **basic Linux administration scripts** written in Bash.
The scripts focus on common **day-to-day operational tasks** performed by Linux
system administrators in server environments.

The goal of this repository is to demonstrate **foundational Linux skills**,
including user management, service monitoring, disk usage checks, and scripting
best practices.

---

## Included Scripts

### user-management.sh
Manages local Linux user accounts.
- Create new users
- Delete existing users
- Designed for administrative use

---

### disk-usage.sh
Checks disk usage on the system.
- Displays filesystem usage
- Helps identify disks nearing capacity
- Useful for proactive monitoring

---

### service-check.sh
Checks the status of a specified systemd service.
- Uses `systemctl`
- Confirms whether a service is running
- Useful for basic service health checks

---

## Use Cases
- Linux server administration
- Operational monitoring
- Learning Bash scripting fundamentals
- Supporting mixed Windows/Linux environments

---

## Technologies & Tools
- Bash
- Linux (systemd-based distributions)
- Core Linux utilities (df, useradd, userdel, systemctl)
